🎯 ДЕМО-СЦЕНАРИЙ ДЛЯ ТЕСТИРОВАНИЯ

   → Должны увидеть приветствие + 6 кнопок ниш

2. Нажмите любую нишу (например "🍕 Ресторан")
   → Бот переключится в режим ассистента ресторана

3. Напишите: "Какое у вас меню?"
   → AI ответит с деталями меню, ценами, рекомендациями

4. Продолжайте диалог (ещё 4 сообщения)
   → После 5-го сообщения появится CTA с предложением покупки

5. Проверьте /reset
   → Должен вернуться к выбору ниши

6. Повторите для других ниш и языков
   → Каждая ниша имеет уникальное поведение

📊 ТЕХНИЧЕСКИЕ ДЕТАЛИ

Архитектура:
- Python 3.11
- aiogram 3.15.0 (async Telegram bot framework)
- Google Gemini 1.5 Flash (AI модель)
- FSM (Finite State Machine) для управления состояниями
- In-memory storage для сессий пользователей

Особенности:
- Определение языка из Telegram profile
- История последних 10 сообщений для контекста
- Персонализированные промпты для каждой ниши
- Graceful error handling с fallback на русский
- Long polling (можно заменить на webhook в production)

Стоимость:
- Gemini API: ~$0.0001 за запрос
- Railway: $5/месяц базовый план
- Telegram Bot: бесплатно
