# AI Centers Demo Bot 🤖

Демо-бот для показа возможностей AI-ассистентов в различных нишах бизнеса.

## 🎯 Возможности

- **6 ниш**: Ресторан, Салон красоты, Доставка, Отель, Фитнес, Другое
- **4 языка**: Русский, English, ქართული, Türkçe (автоопределение)
- **AI-ответы**: Gemini 1.5 Flash
- **Контекстные промпты**: Уникальная роль для каждой ниши
- **CTA**: Автоматическое предложение покупки после 5 сообщений

## 🚀 Деплой на Railway

1. Создайте новый проект на Railway
2. Подключите этот репозиторий
3. Добавьте переменные окружения:
   - `BOT_TOKEN` - токен Telegram бота
   - `GEMINI_API_KEY` - ключ Gemini API
4. Railway автоматически задеплоит бота

## 💻 Локальный запуск

```bash
# Установка зависимостей
pip install -r requirements.txt

# Настройка переменных окружения
cp .env.example .env
# Отредактируйте .env и добавьте свои ключи

# Запуск
python bot.py
```

## 📱 Использование

1. Отправьте `/start` боту
2. Выберите нишу из предложенных кнопок
3. Начните общение с AI-ассистентом
4. После 5 сообщений получите предложение о покупке
